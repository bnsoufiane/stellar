Description

Yukon is a clean HTML5/CSS3 web app based on AngularJS Framework, ui-router (routing framework for AngularJS), Twitter Bootstrap Framework and jQuery, it's easily customisable (less files), cross-browser compatible and works great on mobile devices. There is also HTML/PHP version of Yukon Admin and Landing Page included.

  • One page App (AngularJS)
  • HTML/PHP version
  • Landing Page
  • Responsive Layout (desktops, tablets, mobile devices)

Less

For easy customization Less files are included in Yukon Template. This files are located in assets/less folder.

main.less is the main less file and other less files (mixins, variables, bootstrap style overide, plugins, page specific styles etc) are imported into that one.

After compilation assets/css/main.css and assets/css/main.min.css is created and this is the main stylesheet for Yukon Admin

For more info about less please visit lesscss.org

AngularJS

Yukon Admin is build on AngularJS Framework and ui-router (routing framework), ui-router allows you to organize the parts of your interface into a state machine (please check assets/angular/state.js).

Documetation for ui-roter - https://github.com/angular-ui/ui-router/wiki

All files required for Yukon Admin App is located in - assets/angular/ folder (app.js, controller.js, directive.js, factory.js, service.js, state.js)

AngularJS Directives

pageLoader - page preloader animation <page-loader></page-loader>

menuToggle - side navigation expand/collapse <menu-toggle></menu-toggle>

updateFixedHeaders - update datatables fixedHeader position/size <div update-fixed-headers><table></table></div>

onLastRepeat - ng-repeat after render callback <div ng-repeat="" on-last-repeat></div>

addImageProp - add width/height properities to Image <img ng-src="" alt="" add-image-prop>

Instalation and usage

To install all dependencies go to folder root and use npm install command. This command will download gulp and all required plugins (defined in package.json).

If installation went without errors you can use gulp to:

  • compile less to css ([less] task)
  • minify js ([minify_js] task)
  • combine all js files ([vendor_js] task)
  • reload browser on file change ([default] task, info about browserSync - http://www.browsersync.io/docs/gulp/ )
